# DogQuest

A text-based adventure game built with C++ and SFML.

## Prerequisites

- CMake (version 3.10 or higher)
- C++ compiler with C++17 support
- SFML 2.5 or higher

## Building the Project

1. Create a build directory:
```bash
mkdir build
cd build
```

2. Generate build files:
```bash
cmake ..
```

3. Build the project:
```bash
cmake --build .
```

4. Run the executable:
```bash
./DogQuest
```
